# Content Edit Tool

Tool for editing content files.

## Usage

```bash
# Edit content
./content-edit "content/strategy/article.mdx"

# Edit with editor
./content-edit "content/strategy/article.mdx" --editor "vim"

# Edit metadata
./content-edit "content/strategy/article.mdx" --meta "status:published"
```

## Features

- Content editing
- Metadata editing
- Frontmatter editing
- Editor integration
- Backup creation
- Validation

## Validation

- Content format
- Frontmatter format
- Required fields
- Link validity
- Image references
- Topic references

## Safety Features

- Automatic backup
- Validation checks
- Confirmation prompts
- Dry run mode
- Recovery support
