# Content New Tool

Tool for creating new content files.

## Usage

```bash
# Create content
./content-new

# Create with topic
./content-new --topic "strategy"

# Create with template
./content-new --template "article"
```

## Features

- Content creation
- Topic assignment
- Template support
- Metadata generation
- Path creation
- Validation

## Content Structure

```mdx
---
title: New Article
description: Article description
topic: strategy
date: 2024-03-18
---

# New Article

Content goes here...
```

## Validation

- Title format
- Description length
- Topic validity
- Path uniqueness
- Template validity
- Metadata format
